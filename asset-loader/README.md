## B3 scanner

> Script for fetching CVM reports, pulling data and sending over to remote
> servers.

Running the script
```
    make activate
    b3scan [OPTIONS] <file>...
```

Fetching data
```
    make fetch
```

For help
```
    make help
    b3scan --help
    b3scan --info
```
